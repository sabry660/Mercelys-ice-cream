# Mercelys Ice Cream Website

## Description

A captivating and interactive website for Mercelys Ice Cream, featuring smooth animations, video reels, and a modern design to showcase the art of fine ice cream.

## Features

- Hero section with animated text
- Product details with poetic descriptions
- Marquee highlighting key features: PreBiotic fiber, Natural Color, etc.
- Video reels shared by the community
- Responsive navigation with hamburger menu
- Contact form for distributor enquiries
- Social media links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Locomotive Scroll for smooth scrolling
- GSAP for animations
- Remixicon for icons

## Installation

1. Clone the repository or download the files.
2. Open the project directory.
3. Run a local server to serve the files (since it uses external resources, a server is needed for proper loading).

## Usage

Open index.html in a web browser. The site is optimized for modern browsers.
